# Rollout Templates — チーム展開用のたたき台

> 自分のチーム・会社・プロジェクトに yohakuforce を導入するときに使う **資料テンプレート集** です。
> このフォルダのドキュメントは **そのまま使うものではなく、自組織の文脈に合わせて改変するためのベース** として提供しています。

---

## 何のためのフォルダか

OSS を導入するときに、技術的なセットアップ手順だけでは社内展開は進みません。多くの場合、次のような資料が必要になります。

- 関係者に「これは何か」を 5 分で理解してもらう **概要資料**
- 利用希望者がそれぞれの PC で動かせるようになる **環境構築手順**
- リーダーがチームに見せる **ハンズオン台本**
- 想定される質問への **FAQ**
- 上長や関連部署への **告知文 / アナウンス文**

これらを毎回ゼロから書くのは負担なので、汎用テンプレートとして揃えています。**自分の組織で必要なものだけ抜き出し、`<プレースホルダー>` を自組織の情報に置き換えて使ってください。**

## 使い方

1. このフォルダごと自分のプロジェクトリポジトリにコピー、または個別にコピー
2. 各ファイルの `<プレースホルダー>` を自組織の情報に置換
   - `<チーム名>` `<会社名>` `<実施日>` `<発表者名>` など
3. 不要な節は削除、必要な節は追加
4. 自組織のセキュリティガイドラインや運用ルールに沿った内容に調整

---

## 注意

- これらのテンプレートには **想定値や例示値** が入っています。実際の数値・実績・固有名詞は **必ず自組織のものに差し替えて** ください。
- 顧客名・顧客固有データ・社外秘情報を含めないこと（yohakuforce 自体の禁則ルールとも一致）。
